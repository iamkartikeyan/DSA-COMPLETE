Move to Sorting, Searching, and Hashing. These are asked in almost every interview.

```
04-Sorting-and-Searching/        → All sorts, all searches, advanced search patterns
05-Hashing/                      → Hash tables, frequency counting, rolling hash, Bloom filter
```

Now tackle Trees and Heaps. These test your recursion and structural thinking.

```
06-Trees/                        → Binary Trees, BST, AVL, Heaps, Trie, Segment Tree, Fenwick Tree
```

### 🔴 STEP 4: GRAPHS & DYNAMIC PROGRAMMING (WEEK 7–10)

The heart of interviews. Trees test recursion, graphs test problem modeling, DP separates good from great.

```
07-Graphs/                       → BFS, DFS, Shortest Path, MST, Topological Sort, SCC, Articulation Points
08-Dynamic-Programming/          → Foundations, Knapsack, LCS, Edit Distance, Advanced DP, Optimization
09-Greedy-Algorithms/            → When greedy works, Activity Selection, Huffman Coding, Job Sequencing
10-Divide-and-Conquer/           → Master Theorem, Merge Sort, Quick Sort, Closest Pair
```

### ⚫ STEP 5: ADVANCED PATTERNS (WEEK 11–14)

Finish with everything else — Backtracking, Bit Manipulation, Two Pointers, Sliding Window, and all advanced patterns.

